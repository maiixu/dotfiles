Shared reference for all `obsidian_*` skills. Loaded automatically before any `obsidian_*` operation.

## Vault

- Root: `~/notes/`
- Inbox: `~/notes/收件箱 Inbox/`
- People notes: `~/notes/板块 Areas/人际 Friends/`
- People MOC: `~/notes/板块 Areas/人际 Friends/人际 Friends.md`

## Note Format

Filename: `YYYYMMDDHHmm Title.md`

```
## YYYYMMDDHHmm Title

{body content}

---
YYYYMMDDHHmm

#SubjectTag
#Meta--元数据/Source--来源/Claude-Code

```

Rules:
- H2 heading = ID + space + Title (same as filename without extension)
- `---` separator line after body
- ID repeated on its own line after separator
- Tags one per line (subject tags first, meta last)
- File ends with a space on the last line, then a newline

## Tag System

Naming convention: `#English--中文/Subcategory--中文`

**Meta tags (always at bottom):**

| Tag | Meaning |
|-----|---------|
| `#Meta--元数据/Source--来源/Claude-Code` | Generated by Claude Code |
| `#Meta--元数据/Status--状态/WIP--Work-In-Progress--进行中` | Unfinished note |
| `#Meta--元数据/Type--类型/MOC--Map-of-Content--目录` | Index/navigation page |
| `#Meta--元数据/Type--类型/Flashcards--记忆卡片` | Flashcard note |
| `#Meta--元数据/Type--类型/Collection--合集` | Collection note |

**Common subject tags:**
```
#Productivity--效率/Claude-Code
#Productivity--效率/Claude-Code/Memories--记忆管理
#Productivity--效率/Claude-Code/Skills--技能
#Productivity--效率/PKM--Personal-Knowledge-Management--个人知识管理
#Tech--技术/Obsidian
#Relationship--社交/Notes--备注
```

When in doubt, use a broad parent tag.

## Write Helper

Use Python to write files safely (handles Unicode filenames and content):

```bash
python3 << 'PYEOF'
import pathlib
path = pathlib.Path("{FULL_PATH}")
path.write_text("""{CONTENT}""", encoding="utf-8")
print(f"Saved: {path.name}")
PYEOF
```

## Get Timestamp

```bash
date +"%Y%m%d%H%M"
```

Never use a hardcoded or assumed timestamp.
