## Emoji Conventions in Titles

**Action type:**
- 🔎 = Research / investigate
- 🔍 = Deep strategy research (especially investments)
- 💻 = Tech/computer action
- 📱 = Mobile/app action
- 📞 = Phone call needed
- 💬 = Waiting for reply / communication
- 🏗️ = Ongoing build project
- 💭 = Blog post / essay idea
- ✅ = Daily review/check-in routine

**Content type:**
- 🥚 = Meal (recurring nutrition task)
- 🍅 = Pomodoro/timed routine (e.g. after-meal routine)
- 📖 = Book to read
- 🎸 = Guitar practice session
- 🍿 = Movie/TV premiere
- 🗓️ = Calendar event or trip

**Financial:**
- 🌟 = Credit card reward / gift card to use
- 💰 = Money transfer / financial action
- 💳 = Purchase / credit card task

**Status:**
- 🔴 Blocked = tag on tasks blocked by something
- [Blocked by X] = inline dependency notation in title

## Tagging Conventions

**Duration (time estimation):**
- `5m`, `10m`, `15m`, `30m`, `45m`, `1h`, `1h15m`, `1h30m`, `1h45m`, `2h`, `4h`, `8h`

**Energy level:**
- `🔋 High Voltage` = high focus/energy required
- `🪫 Low Voltage` = low-energy task
- `⚡️ Energy` = general energy tag

**Time of day:**
- `🪥 Morning` = morning routine
- `🧴 Evening` = evening routine

**Recurring frequency:**
- `Daily`, `Weekly`, `Biweekly`, `Monthly`, `Quarterly`, `Yearly`
- `Every Three Weeks`, `Every Five Weeks`, etc.
- `Once Every Two Days`, `Once Every Four Days`
- `🗓️ Agenda` = agenda item for a meeting

**Other:**
- `🛒 Errands` = physical errands
- `📖 Reading List` = to-read item
- `📝 Template` = task template
- `⏰ Estimated` = has time estimate
- `🌟 Starred` = starred/important
- `On Demand` = trigger-based, not scheduled
